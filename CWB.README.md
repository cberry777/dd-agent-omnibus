# Merge to upstream

```bash
$ git remote add upstream https://github.com/DataDog/dd-agent-omnibus.git
$ git remote -v
origin	https://github.com/cberry777/dd-agent-omnibus.git (fetch)
origin	https://github.com/cberry777/dd-agent-omnibus.git (push)
upstream	https://github.com/DataDog/dd-agent-omnibus.git (fetch)
upstream	https://github.com/DataDog/dd-agent-omnibus.git (push)
$ git pull upstream master
```
